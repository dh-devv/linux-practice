# 리눅스 기초 명령어 학습

## 1. 기본 명령어
* `ls` : 파일 보기
* `pwd` : 현재 나의 위치 확인
* `cd` : 디렉터리 이동
* `cd ..` : 상위 디렉터리로 이동

---

## 2. 시스템 종료 및 재부팅
* `init 0` : 시스템 종료
* `init 6` : 시스템 재부팅
* `shutdown -h` : 시스템 종료
* `shutdown -k` : 종료 알림
* `shutdown -c` : 종료 취소
* `reboot` : 시스템 재부팅
* `poweroff` : 전원 종료

---

## 3. 실습 코드

```bash
ls -l
cd /etc
pwd
```
