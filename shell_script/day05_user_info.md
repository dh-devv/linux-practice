# 사용자 정보 출력 셸 스크립트

현재 로그인한 사용자의 이름과 홈 디렉터리를 출력하는 간단한 셸 스크립트 실습입니다.

## 파일 생성

```bash
gedit name.sh
```

---

## 실습 코드

```sh
#!/bin/sh

echo "사용자 이름: "  $USER
echo "홈 디렉터리: "  $HOME

exit 0
```

---

## 실행 방법

### 방법 1. sh 명령어로 실행한다.

```bash
sh name.sh
```

---

### 방법 2. 실행 권한을 부여한 후 실행한다.

```bash
chmod +x name.sh
./name.sh
```

---

## 실행 결과

```text
사용자 이름: root
홈 디렉터리: /root
```

---

## 코드 설명

- `#!/bin/sh` : 셸 스크립트를 실행한 셸을 지정한다.
- `echo` : 출력할 내용을 적는 명령어이다.
- `$USER` : 현재 로그인한 사용자 이름을 저장한 변수이다.
- `$HOME` : 현재 사용자의 홈 디렉터리를 저장한 변수이다.
- `exit 0` : 프로그램을 정상적으로 종료한다.
